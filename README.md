## Запуск проекта

### Установить зависимости

```sh
$ npm i
```

### Просмотр

```sh
$ npm start
```
Проект будет доступен на [localhost:3000](http://localhost:3000/)

### Сборка для продакшн

```sh
$ npm run build
```
Результат в папке `dist`
